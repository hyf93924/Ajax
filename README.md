##### ajax对象要分IE系列浏览器和非IE系列浏览器    
- IE系列：存在ActiveXObject就是IE系列的浏览器，实现起来分IE版本比肩复杂   
- 非IE系列：存在XMLHttpRequest就是非IE系列浏览器    

---
